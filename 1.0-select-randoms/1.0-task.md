Челендж 1.0. Простые числа
==========================

Есть xml такого вида:
    <items>
        <item>...</item>
        <item>...</item>
        <item>...</item>
        ...
        </item>...</item>
    </items>
Требуется написать xsl, выбирающий сколько-то случайных item'ов из этого списка.

Например, из такого списка:
    <items>
        <item>3</item>
        <item>7</item>
        <item>1</item>
        <item>2</item>
        <item>8</item>
        <item>6</item>
        <item>9</item>
        <item>4</item>
        <item>5</item>
    </items>
выбрать 5 случайных.
